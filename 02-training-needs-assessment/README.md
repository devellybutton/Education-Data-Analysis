## 1. 프로젝트 개요

### 배경  
- 스마트팩토리 전환을 추진 중인 제조 기업을 대상으로  
맞춤형 교육을 운영하던 과정에서 **공통 교육과정 개설 필요성**이 발생했다.  
- 그러나 과정 우선순위를 판단할 **정량적 기준이 없는 상태**였다.

### 목표  
- 기업 HRD 담당자 설문 데이터를 기반으로  
교육 과정별 **중요도와 실행도 구조화**,  
그리고 **공통 교육과정 개설 우선순위 도출**을 한다.

### 분석 대상  
- 제조 기업 **HRD 담당자 설문 응답** 60건

## 2. 문제 정의 및 분석 관점

스마트팩토리 전환 기업을 대상으로 공통 교육과정을 설계해야 하는 상황에서 <br/> 
다음과 같은 **운영 의사결정 문제**가 존재했다.

- 어떤 교육 과정을 **우선적으로 개설해야 하는가**  
- 현장 요구와 현재 실행 수준 간 **차이는 어디에서 발생하는가**  
- 중요하게 인식되지만 충분히 제공되지 않는 과정은 무엇인가  

본 분석은 교육 과정별 <b>중요도(Importance)</b>와 <b>실행도(Performance)</b>를 기준으로  
요구 구조를 정량적으로 비교하고, **중요도–실행도 GAP**을 기반으로  
**공통 교육과정 개설 우선순위 도출**을 목표로 했다.

분석에는 다음 기준을 적용했다.

- 중요도 및 실행도 점수 기반 **정량 구조화**  
- **중요도–실행도 GAP 계산**  
- GAP 패턴을 기준으로 **교육 과정 유형 분류**  
- 유형별 **우선순위 도출**
