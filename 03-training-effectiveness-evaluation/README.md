## 1. 프로젝트 개요

### 1-1. 배경

- 교육 센터에서는 매 회차 교육을 운영하고 있었으나,
교육의 실제 효과를 정량적으로 입증할 수 있는 지표가 부족한 상황이었다.
- 기존에는 만족도 평균 점수만 확인하고 있었으며,
"교육이 실제로 역량 향상에 기여했는지"에 대한 통계적 검증은 이루어지지 않았다.
- 이에 **동일 참여자**를 대상으로 **교육 전·후 설문**을 실시하여
교육 효과를 정량적으로 검증하고자 하였다.

### 1-2. 목표

- 동일 집단의 교육 전·후 점수 차이가 통계적으로 유의한지 검증한다.
- 교육 효과의 실질적 크기(Effect Size)를 산출한다.
- 어떤 조건(회차, 직종, 사용 경험 등)에서 교육 효과가 더 크게 나타나는지 분석한다.
- 향후 교육 개선을 위한 데이터 기반 의사결정 근거를 마련한다.

### 1-3. 데이터

- 1개월 간 오프라인 교육 참여자 설문 **100건 (5회차)**
    - 동일 참여자의 사전·사후 설문 매칭 데이터
    - 리커트 5점 척도 기반 역량 문항 (3지표 × 2문항)
    - 직종, 엑셀 사용 경험, 회차 정보 등 보조 변수 포함

## 2. 문제 정의

교육 운영 현장에서는 다음과 같은 현실적 제약이 존재한다.

- 응답자의 오기재 (회차, 날짜, 직종 등)
- 교육 전 또는 교육 후 설문 미완료로 인한 결측 발생
- 중복 응답
- 특정 회차의 낮은 응답률

이러한 조건 속에서 다음을 검증해야 했다.

- 교육 전·후 동일 집단에서 유의미한 점수 차이가 존재하는가?
- 단순 평균 상승이 아닌, 통계적으로 검증 가능한 효과인가?
- 특정 집단에서 교육 효과가 더 크게 나타나는가?

## 3. 가설 설정

| 순번 | 분석 관점 | 가설 | 분석 목적 |
|------|-----------|------|-----------|
| H1 | 전체 효과 | 교육 후 평균 점수는 교육 전보다 유의하게 높다 | 교육 효과 존재 여부 검증 |
| H2 | 효과 크기 | 교육 효과는 실질적으로 의미 있는 수준이다 (Cohen's d ≥ 기준값) | 단순 유의성 넘어서 실질적 영향 평가 |
| H3 | 집단 차이 | 직종/회차/사용경력에 따라 향상폭에 차이가 있다 | 교육 효과의 조건 탐색 |
| H4 | 예측 요인 | 초기 점수 및 배경 변수는 향상폭에 영향을 미친다 | 향상 요인 구조 분석 |

## 4. 데이터 및 지표 정의

### 4-1. 동일 집단 통제 방법

- 사전·사후 설문에 고유 식별 키 부여 (익명 ID)
- 사전·사후 응답 매칭 후 불일치 데이터 제거
- 사전·사후 모두 응답한 참여자만 분석 대상 포함

<br/>

### 4-2. 지표 정의

| 구분 | 지표 |
|------|------|
| 기본 지표 | 사전 평균 점수 / 사후 평균 점수 / 평균 향상폭 (post - pre) |
| 통계 검증 | 대응표본 t-test / Wilcoxon 검정 / p-value / 95% 신뢰구간 |
| 효과 크기 | Cohen's d (0.2=소, 0.5=중, 0.8 이상=대) |
| 분포 관련 | 사전 점수 분포 / Ceiling Effect 여부 확인 |
| 세그먼트 | 회차별 / 직종별 / 엑셀 사용경력별 평균 향상폭 |

## 5. 분석/검증

### 5-1. 가설 검증
| 가설     | 분석 관점 | 핵심 검정 결과                                                               | 결론           | 해석                                         |
| ------ | ----- | ---------------------------------------------------------------------- | ------------ | ------------------------------------------ |
| **H1** | 전체 효과 | 대응표본 t-test: p < .001<br>Wilcoxon: p < .001<br>95% CI: (0.826 ~ 0.972) | **지지**     | 교육 후 평균 점수는 통계적으로 유의하게 상승함                 |
| **H2** | 효과 크기 | Cohen’s d = 2.65 (0.8 초과-큼)                                     | **강하게 지지** | 효과는 통계적으로뿐 아니라 실질적으로도 매우 큼                 |
| **H3** | 집단 차이 | 엑셀경력 ANOVA: p < .001<br>직무 ANOVA: p = 0.896<br>회차 ANOVA: p = 0.301     | **부분 지지** | 엑셀 사용경력에 따라 향상폭 차이 존재<br>(낮을수록 향상폭이 큼)<br>직무/회차 차이는 유의하지 않음 |
| **H4** | 예측 요인 | OLS 회귀: p < .001<br>R² = 0.341<br>계수 = -0.335                          | **지지**     | 초기 점수가 낮을수록 향상폭이 큼 (음의 관계)                 |

## 6. 인사이트 해석

1. 교육은 전반적으로 통계적으로 유의한 효과를 보였다.
   - 전후 평균 차이가 유의하며,
   - 중간 이상의 효과 크기를 확인하였다.

2. 초기 점수가 낮은 학습자일수록 향상폭이 더 크게 나타났다.
   - 사전 점수와 향상폭 간 음의 관계(β = -0.335).
   - 교육은 기초 역량이 낮은 집단에서 특히 효과적이다.

3. 회차 간 향상폭 차이는 크지 않았다.
   - 그래도 차이가 있으니, 교육 운영의 일관성을 더 유지해야할 수 있다. 

4. 엑셀 사용 경험이 적은 집단에서 상대적으로 높은 개선 효과가 관찰되었다.
   - 실습 기반 교육이 초급자에게 높은 체감 효과를 제공한 것으로 해석된다.
